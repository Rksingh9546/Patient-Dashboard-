# Patient Dashboard — Coalition Technologies FED Skills Test

A responsive React-based patient dashboard built from the provided Adobe XD design using the Coalition Technologies Patient Data API.

---

## Features

- Adobe XD to React conversion
- Responsive dashboard layout
- API integration using Axios
- Basic Authentication implementation
- Blood Pressure Chart using Chart.js
- Patient Profile Section
- Diagnosis History
- Diagnostic List Table
- Lab Results Section
- Modern healthcare dashboard UI

---

## Technologies Used

- React.js
- Vite
- Axios
- Chart.js
- React ChartJS 2
- CSS3
